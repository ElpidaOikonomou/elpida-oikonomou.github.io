# elpida-oikonomou.github.io

Personal CV website for Elpida Oikonomou.

## Quick start
1. Install dependencies:
```bash
npm install
```
2. Run the dev server:
```bash
npm run dev
```
3. Build:
```bash
npm run build
npm start
```

## Deploy to Vercel
Push this repo to GitHub and connect it to Vercel. Vercel auto-detects Next.js.
